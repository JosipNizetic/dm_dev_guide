################################
NCSA Nebula OpenStack User Guide
################################

Nebula is an OpenStack system available to LSST DM developers.
OpenStack allows you to work in a cloud environment with flexible compute instances and storage.

To request an account on the Nebula system, `file a JIRA ticket <https://jira.lsstcorp.org/secure/CreateIssueDetails!init.jspa?pid=12200&issuetype=10902&priority=10000>`_ in the IT Helpdesk Support (IHS) project.
Include the following in the request:

   - First and last name
   - Email
   - Phone number
   - Mailing address
   - Sponsoring LSST manager
   - LSST Team or project
